About the project

The Library Management System will be an application that focuses on helping libraries manage their collection of books as well as making the library experience much easier and streamlined for the application's main demographic: the library visitors.

Goals

The application will be able to help users find and access books found in the library. Its features shall encourage more people to go to libraries and read books. The library management system will accomplish this via its user-friendly UI and the features included in the application, allowing users to who have never visited a library before to be just as prepared as someone who is already very familiar with them.

Features

The library management system will have a search function that allows users to find a specific book they are looking for or filter by distinct categories and genres. Each book will have information stored on it on the database. This information will include, but is not limited to, a description of the book, author, date of publishing and genre. The application will also feature a map so that users can better understand the layout of the library.

Details

Libraries already keep a database with information on all their books, which is why the Library Management System will use the already existing database rather than creating its own. Although the application for now will be created based on its ability function in a single library, using a library's already existing database will help with making the application work with many distinct libraries.

In addition, the Library Management System will not manage borrowing books, the creation of Library memberships, nor the payment for late fees. The application will, however, have access to this information in order to notify the user that their due date for returning the book is approaching.
